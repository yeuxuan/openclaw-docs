# macOS App 首次启动指南

> 这是 **方式一：macOS 桌面应用** 的安装指南。
> 如果你想用命令行安装，请看[命令行向导安装指南](./wizard)。

跟着下面的步骤完成首次启动，全程大约 5～10 分钟。

**还没有下载？** 前往 [GitHub Releases](https://github.com/openclaw/openclaw/releases) 页面，下载最新版本的 `.dmg` 文件，双击安装后再继续。

**开始之前，请确认你已经有了 AI API 密钥。**
还没有？→ [回到快速开始，先获取 API 密钥](./getting-started#第一步准备-api-密钥)

---

## 第一步：绕过 macOS 安全警告

双击打开应用时，可能会弹出这个提示：

```
"OpenClaw" 无法打开，因为苹果无法检查其是否包含恶意软件。
```

**这是正常的！** macOS 对所有没有通过 App Store 的应用都会显示这个警告，不代表软件有问题。

处理方法：

1. 点击弹窗里的 **"取消"**（不要点"移到废纸篓"）
2. 打开 **系统设置**（或系统偏好设置）→ **隐私与安全性**
3. 向下滚动，找到关于 OpenClaw 的提示，点击 **"仍要打开"**
4. 再次确认打开

---

## 第二步：允许访问本地网络

应用启动后会弹出权限请求：

```
OpenClaw 想要找到和连接本地网络上的设备
```

点击 **"允许"**。这是必须的权限，OpenClaw 需要它来在你的局域网内正常工作。

---

## 第三步：阅读欢迎说明

打开后会出现欢迎界面，里面说明了 OpenClaw 能做什么、有哪些注意事项。**建议花 1 分钟看一下**，了解基本概念后操作更顺畅。

读完后点击 **"继续"**。

---

## 第四步：选择网关运行位置

```
在哪里运行 Gateway（网关）？
● 这台 Mac（推荐）
○ 远程服务器（通过 SSH / Tailscale）
○ 稍后配置
```

> **网关是什么？**
> 网关是 OpenClaw 的"指挥中心"，所有消息都经过它。它必须持续运行，AI 才能接收和回复你的消息。

**选择"这台 Mac"（推荐）**，让网关就运行在你眼前这台电脑上，最简单。

---

## 第五步：填入 AI API 密钥

这是最重要的一步。

1. 选择 AI 提供商，推荐选 **Anthropic（Claude）**
2. 在输入框里粘贴你的 API 密钥（`sk-ant-` 开头那串字符）
3. 点击确认

::: tip 还没有 API 密钥？
1. 访问 [console.anthropic.com](https://console.anthropic.com)
2. 注册并登录
3. 点击 "API Keys" → "Create Key"，复制密钥（只显示一次！）
:::

---

## 第六步：授予系统权限

应用会请求一些可选权限，按需开启即可：

| 权限 | 用途 | 建议 |
|------|------|------|
| 通知 | 任务完成时提醒你 | 建议允许 |
| 辅助功能 | 让 AI 帮你操控其他应用 | 需要自动化时再开 |
| 麦克风 | 语音输入 | 按需 |
| 摄像头 | 图像分析 | 按需 |
| 屏幕录制 | 截图功能 | 按需 |

> 不确定要不要开？**先只允许"通知"**，其他的需要时随时可以在系统设置里补开。

---

## 第七步：安装命令行工具（推荐）

```
是否安装 OpenClaw CLI（命令行工具）？
● 是（推荐）
○ 否
```

**推荐选"是"**。安装后，你可以用终端命令来管理 OpenClaw，比如查看状态、重启网关等。

---

## 第八步：AI 助手自我介绍

配置完成后，应用会自动打开一个对话窗口，你的 AI 助手会主动和你打招呼，介绍它能做什么。

**如果 AI 回复了你——恭喜！OpenClaw 已经成功运行！**

---

## 启动后：菜单栏图标

成功后，macOS 菜单栏右上角会出现一个 OpenClaw 图标。点击它可以：

- 查看 AI 服务状态（运行中 / 停止）
- 快速打开控制面板
- 重启或停止网关

---

## 常见问题

::: details 菜单栏里没有看到图标？
可能是应用没有启动成功。

1. 在 Finder 里找到 OpenClaw.app，双击重新打开
2. 如果出现安全警告，按照第一步的方法处理
:::

::: details AI 助手没有回复？
按顺序检查：

1. 菜单栏图标是否正常显示（不是红色）
2. API 密钥是否正确（可以在应用设置里重新填写）
3. 网络是否能访问 Anthropic / OpenAI（中国大陆可能需要代理）
:::

::: details 怎么在 Telegram 里使用 AI？
需要连接"通道"。→ [查看 Telegram 接入教程](../channels/telegram)
:::
